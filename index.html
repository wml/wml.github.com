Yodle, Inc.

My cross-functional team of five software engineers delivered several key digital marketing innovations, among them mobile-optimized landing pages, a website-integrated mobile review solicitation app for Android and iPhone, a Facebook-integrated social promotions widget, and a restructuring of Yodle's core product offerings including the addition of a low-cost, highly-performant entry-level SEO product. Bank of America Merrill Lynch

In order to tackle the issue of lost revenue and trade expediency due to client connectivity misconfigurations, I led a small team of java engineers in designing and implementing a centralized configuration management system. The connectivity system pushed much of the complexity usually found in the application layer down into XML configuration files which were edited by hand and deployed on an ad-hoc basis. This error-prone process regularly resulted in costly downtime for clients. The new system provides an easy-to-use GUI for editing client configurations, validation, and a standardized, auditable deployment mechanism. With the first release alone, the operational losses realized from misconfigurations were reduced ten-fold.

Hillier Architecture

As an in-house developer, I created several applications in support of the planning, execution and monitoring of large architecture projects. The most innovative project was an interactive Punchlist application designed for the first generation of Windows tablets. Architects would leverage the app's video-capture, speech and handwriting recognition to document construction flaws in rugged external environments where a traditional keyboard would be difficult to use, automatically generating a report from the data collected on-site upon return to the office.
